QueryRequest
============

A ``QueryRequest`` is used to perform a ``query`` transaction with DynamoDB. It is used for retrieving one or many items by its primary key. The partition key is always required.

https://docs.aws.amazon.com/amazondynamodb/latest/APIReference/API_Query.html#DDB-Query-request-KeyConditionExpression
